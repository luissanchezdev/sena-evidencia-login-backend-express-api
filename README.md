## Evidencia SENA - API con Express para login


- Instalar dependencias
```bash
npm install
```

- Configurar variables de entorno

```bash
PORT = 4000
DB = 'NOMBRE_BASE_DE_DATOS'
DB_USER = 'NOMBRE_USUARIO_BASE_DE_DATOS'
DB_PASSWORD = 'CONTRASENIA_BASE_DE_DATOS'
```

- Iniciar el proyecto
```
npm run dev
```
